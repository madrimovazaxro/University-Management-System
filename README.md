# University-Management-System
C++ OOP project demonstrating inheritance, polymorphism, and memory-safe design using a Student &amp; Teacher management system with a CLI menu.

# 🎓 Person Management System (C++ OOP)

This project is a console-based application written in C++ that demonstrates core Object-Oriented Programming (OOP) concepts through a simple Person management system.

The system models a base class `Person` and two derived classes: `Student` and `Teacher`. It allows users to dynamically create and manage records via a menu-driven interface.

## 🚀 Features

- Add new Students and Teachers  
- Display all stored records  
- Input validation (age, gender, grade, ID)  
- Clean console output formatting  
- Menu-driven interaction  

## 🧠 OOP Concepts Demonstrated

- **Encapsulation** – controlled data setting via methods  
- **Inheritance** – `Student` and `Teacher` extend `Person`  
- **Polymorphism** – virtual `show()` function with overrides  
- **Abstraction** – base class interface design  

## ⚙️ Technical Highlights

- Uses `std::vector` for dynamic storage  
- Implements **smart pointers (`std::unique_ptr`)** for safe memory management  
- Modular structure with header and source files  
- Input validation and error handling
